# Media Search Web App

A responsive web application that allows users to search and explore high-quality photos and videos instantly. Users can browse, save, and download media with a smooth and modern UI experience.

---

## Features

- Real-time search functionality
- High-quality photo results
- Video preview support
- Save items to personal collection (using localStorage)
- Download photos and videos
- Separate tabs for Photos and Videos
- Responsive and modern UI design
- Hover effects and interactive cards

---

## Tech Stack

- React.js
- Redux Toolkit
- Tailwind CSS
- JavaScript (ES6+)
- REST APIs (for media data)
- LocalStorage (for saved collection)

## How It Works

1. User enters a search query
2. App fetches data from external APIs
3. Results are displayed as photos or videos
4. User can switch between Photos and Videos tabs
5. User can save items to collection (stored in localStorage)
6. User can download media files directly
